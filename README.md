# Happy

## Front-End:
React.js, HTML, CSS e TypeScript, biblioteca leaflet para a utilização de mapas na aplicação.

## Back-End: 
Feito em node.js + express, TypeORM, validações no react com o yup, e banco de dados SQLite.
